# 🏥 MedBot — AI Sales Manager

Телеграм-бот, эмулирующий менеджера по продажам медицинских услуг.
Бот квалифицирует лида (медицинский запрос или болтовня) и использует разные стратегии общения.

## 🛠 Технологии
- **Python 3.12**
- **aiogram 3.x** — асинхронный фреймворк для Telegram.
- **OpenAI API (GPT-4o-mini)** — основной движок (ветка `main`).
- **Pydantic v2** — для валидации и Structured Outputs.
- **uv** — современный менеджер пакетов.

## 🚀 Установка и запуск

Проект управляется через `uv`.

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/твое-имя/med-bot.git
   cd med-bot
   ```

2. **Настройте переменные окружения:**

    Создайте файл `.env`
    ```env
    BOT_TOKEN=ваш_токен_от_botfather
    OPENAI_API_KEY=sk-proj-...
    OPENAI_MODEL=gpt-4o-mini
    ```

3. **Установите зависимости и запустите:**
   Сначала синхронизируем окружение (установим библиотеки из lock-файла):
   ```bash
   uv sync

4. **Запуск бота:**
    ```bash
