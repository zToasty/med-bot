# 🏥 MedBot — AI Sales Manager

Телеграм-бот, эмулирующий менеджера по продажам медицинских услуг (Елена). 
Бот отвечает на вопросы пациентов, показывает цены, присылает фото до/после и записывает на консультации через Google Таблицы.

## ✨ Основные фичи
- **Tool Calling (OpenAI)** — бот сам решает, когда искать по базе, когда отправлять фото, а когда записывать в слот.
- **RAG (ChromaDB)** — поиск ответов по медицинской базе знаний и отзывам.
- **Персистентная память** — история диалогов сохраняется в локальной SQLite базе (`bot_data.db`), бот не забывает контекст при перезапуске.
- **Умная запись (gspread)** — интеграция с Google Sheets для проверки свободных слотов и записи пациентов. Встроена защита от двойной записи.
- **Безопасность** — жесткие правила (anti-discount) и защита от промпт-инъекций. Запрещено менять цены.
- **Файловое логирование** — сессии логгируются в папку `logs/`.

## 🛠 Технологии
- **Python 3.12+**
- **aiogram 3.x** — асинхронный фреймворк для Telegram.
- **OpenAI API (gpt-4o-mini)** — основной LLM-движок.
- **ChromaDB** — векторная база для RAG.
- **gspread** — работа с Google Таблицами (асинхронно через `asyncio.to_thread`).
- **SQLite** — встроенная база для хранения истории.
- **uv** — современный сверхбыстрый менеджер пакетов.

## 🚀 Установка и запуск

Проект управляется через `uv`.

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/zToasty/med-bot.git
   cd med-bot
   ```

2. **Настройте переменные окружения:**

   Создайте файл `.env` в корневой папке проекта:
   ```env
   BOT_TOKEN=ваш_токен_от_botfather
   OPENAI_API_KEY=sk-proj-...
   OPENAI_MODEL=gpt-4o-mini
   GOOGLE_SHEET_ID=айди_гугл_таблицы
   ADMIN_CHAT_ID=ваш_тг_айди_для_уведомлений
   ```

3. **Google Service Account (для записи)**
   Поместите файл `creds.json` (ключ сервисного аккаунта Google) в корень проекта. У этого аккаунта должен быть доступ к таблице `GOOGLE_SHEET_ID`.

4. **Установите зависимости и обновите базу знаний:**
   ```bash
   uv sync
   uv run crawler/run.py
   ```

5. **Запуск бота:**
   ```bash
   uv run main.py
   ```
   *Все логи будут выводиться в консоль и параллельно сохраняться в директорию `logs/`.*
